Microsoft Office

Microsoft Office is a suite of applications that are helpful in day-to-day use with the computer.

Microsoft Office Applications Desctiptions

Word: A word processing application that allows you to create a variety of documents, including letters, resumes, and more.

PowerPoint: A presentation program that allows you to create dynamic slide presentations.

Excel: An electronic spreadsheet is a computer software program that is used for storing, organizing, and manipulating data.

These applications are always being updated to keep up with todays growing technology.
